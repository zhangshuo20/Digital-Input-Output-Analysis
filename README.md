# Digital-Input-Output Analysis

This repository provides the data and code used for the digital input-output analysis in the manuscript. The analysis quantifies how core digital products contribute to value added embodied in household consumption and decomposes this contribution into supply-chain support effects and direct final-demand effects.
## Repository contents
- `Digital SUTs and IOTs.zip`  
  Original digital supply-use tables and input-output tables.

- `Digital IO Tables 2000-2020 for China.xlsx`  
  Digital input-output tables for China from 2000 to 2020 used in this study.

- `main.py`  
  Core script for calculating value added embodied in household consumption and decomposing the contribution of core digital products into supply-chain support and direct final-demand effects.

- `Robustness_Narrow_DP.py`  
  Robustness analysis using a narrower definition of core digital products.

- `Robustness_digital_input_intensity.py`  
  Robustness analysis using alternative specifications of digital input intensity.

- `Robustness_substitution.py`  
  Robustness analysis allowing partial substitution of removed core digital inputs by non-digital suppliers.

- `supplementary_transmission_analysis.ipynb`  
  Supplementary transmission analysis used to calculate and plot Figure 5b, 5d, 5f, 5h, and Figure 6.

- `plot.ipynb`  
  Notebook for plotting Figures 2 and 3.

- `plot_sector.ipynb`  
  Notebook for plotting Figure 4 and Figure 5a, 5c, 5e, and 5g.

- `plot_Robustness_Narrow_DP.ipynb`  
  Notebook for plotting Figure S1.

- `plot_robustness_digital_input_intensity.ipynb`  
  Notebook for plotting Figures S2 and S3.

- `plot_robustness_substitution.ipynb`  
  Notebook for plotting Figure S4.
